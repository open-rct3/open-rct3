# How to Contribute to OpenRCT3

## 🐛 Found a bug?

1. Confirm whether the issue already exists by searching existing
   [Issues](https://github.com/open-rct3/open-rct3/issues?q=is%3Aissue).
2. If the problem is new, open a [new issue](https://github.com/open-rct3/open-rct3/issues/new).

Please include a succinct title and a clear description, including as many details as possible so that we can reproduce,
fix, and resolve your issue.

## Contributing Code

See the [Debugging](./docs/Debugging.md) guide.

### 🛠️ Fixing a bug?

1. Confirm an
   [open issue](https://github.com/open-rct3/open-rct3/issues?q=is%3Aissue%20state%3Aopen%20-type%3AFeature%20-type%3ATask)
   exists for the problem.
2. Open a [new draft pull request](https://github.com/open-rct3/open-rct3/compare) (PR). Ensure the PR description
   clearly describes the problem and solution. Include and link to the issue number, if applicable.
3. Request a code review from one of the project's [maintainers](https://github.com/open-rct3/open-rct3/people).
4. [Iterate](https://en.wikipedia.org/wiki/Iterative_and_incremental_development)!

### ✨ Implementing a new feature?

1. Open a [new suggestion](https://github.com/open-rct3/open-rct3/discussions/new?category=ideas) to discuss the changes
   you'd like to see, [fork](https://github.com/open-rct3/open-rct3/fork) the repo, create a new feature branch, and
   start making changes.
2. Open a [new draft pull request](https://github.com/open-rct3/open-rct3/compare).
3. Request a code review from one of the project's [maintainers](https://github.com/open-rct3/open-rct3/people).
4. [Iterate](https://en.wikipedia.org/wiki/Iterative_and_incremental_development)!

## Questions

Feel free to ask any question in the [Q&A forum](https://github.com/open-rct3/open-rct3/discussions/categories/q-a) or
on [Reddit](https://www.reddit.com/r/OpenRCT3).

## Other Ways to Help

Please feel free to share your ideas with us on [Reddit](https://www.reddit.com/r/OpenRCT3) or on the
[official forum](https://github.com/open-rct3/open-rct3/discussions).

Check out the [Roadmap](https://github.com/open-rct3/open-rct3/wiki/Roadmap), open a new discussion on the
[forum](https://github.com/open-rct3/open-rct3/discussions) (GitHub Discussions), or
[buy us a coffee](https://github.com/sponsors/chances). Every bit of your support helps us make this dream (some say
pipe-dream) a reality!
