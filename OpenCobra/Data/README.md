# OpenRCT3 libData

C# library to read and write RCT3's non-OVL DAT container format, used by saved parks
(`Documents\RCT3\Parks\*.dat`), coaster track designs (`*.trk`), and firework files (`*.fwd`,
`*.frw`). Distinct from the OVL archive format read by [`OpenCobra.OVL`](../OVL#readme).

## References

- [belgabor.vodhin.org/format](http://belgabor.vodhin.org/format/): Documentation of the non-OVL file format.
